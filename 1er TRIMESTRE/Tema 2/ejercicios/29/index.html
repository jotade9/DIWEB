Overflow: hidden;

Este es un ejemplo con un contenido largo. Si no ves toda la información es porque el contenedor tiene overflow: hidden y se corta cualquier contenido que se salga de los límites.

Overflow: scroll;

Este es un ejemplo con un contenido largo. Aquí verás barras de desplazamiento en ambas direcciones (vertical y horizontal) porque el contenedor tiene overflow: scroll.

Overflow: auto;

Este es un ejemplo con un contenido largo. Si el contenido se desborda, aparecerán las barras de desplazamiento solo cuando sea necesario, ya que el contenedor tiene overflow: auto.

Overflow-X: scroll; Overflow-Y: hidden;

Este es un ejemplo con contenido muy ancho. Solo aparecerá una barra de desplazamiento horizontal, ya que hemos configurado overflow-x: scroll y overflow-y: hidden.

Overflow-X: hidden; Overflow-Y: scroll;

Este es un ejemplo con contenido muy largo. Solo aparecerá una barra de desplazamiento vertical, ya que hemos configurado overflow-x: hidden y overflow-y: scroll.
